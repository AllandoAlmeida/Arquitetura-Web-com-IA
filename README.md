# 🏗️ Arquitetura Web com IA

Projeto desenvolvido ao longo de uma trilha de 5 aulas, com o objetivo de entender
**na prática** como aplicações web modernas funcionam de ponta a ponta — do clique
do usuário até a resposta do servidor.

## 🎯 Sobre o projeto

Este repositório documenta a construção de uma aplicação funcional, explorando
conceitos fundamentais de arquitetura de software com o apoio de Inteligência
Artificial como ferramenta de desenvolvimento.

## 📚 Conceitos abordados

- **Frontend** — interface e experiência do usuário
- **Backend** — lógica de negócio e processamento de dados
- **HTTP** — protocolo de comunicação entre cliente e servidor
- **Modelo Client-Server** — como as partes se conectam e trocam informações
- **Separação de Responsabilidades** — organização e boas práticas de arquitetura

## 🤖 Desenvolvimento com IA

Ao longo do projeto, a IA foi utilizada como parceira de desenvolvimento,
auxiliando na escrita de código, tomada de decisões arquiteturais e resolução
de problemas — refletindo um fluxo de trabalho cada vez mais comum no
desenvolvimento de software moderno.
